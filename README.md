# 🎮 推箱子 Sokoban

一个经典的推箱子（Sokoban）网页小游戏，纯前端实现，无需后端。

## 🎯 功能

- 7 个精心设计的关卡，难度从入门到大师
- 📖 每关配有详细教程（攻略提示、核心概念、步骤指导）
- ⌨️ 键盘操作（方向键/WASD）
- 📱 移动端支持（触摸滑动 + 虚拟方向键）
- ↩️ 撤销 (Z) / 🔄 重置 (R) / 📖 教程 (T)
- 📋 关卡选择界面
- 💾 自动存档（localStorage）
- 🎨 精美的 Canvas 绘制界面

## 🚀 运行

直接用浏览器打开 `index.html` 即可，无需安装任何依赖。

```bash
# 或者用 Python 起个本地服务器
python3 -m http.server 8080
# 然后打开 http://localhost:8080
```

## 🎮 操作说明

| 操作 | 键盘 | 手机 |
|------|------|------|
| 移动 | 方向键 / WASD | 滑动 / 虚拟方向键 |
| 撤销 | Z | 按钮 |
| 重置 | R | 按钮 |
| 教程 | T | 按钮 |
| 选关 | 点击按钮 | 点击按钮 |
